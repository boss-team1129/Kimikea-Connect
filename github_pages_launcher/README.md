# Kimikea Connect Launcher

このフォルダは過去に作成したランチャー用ファイルの保管場所です。

現在のKimikea Connect本番入口URLは以下に統一します。

https://boss-team1129.github.io/Kimikea-Connect/

## 運用ルール

- 加盟店・コーディネーターへ案内するURLは正式入口URLだけにする
- このランチャーURLやApps Script URLは加盟店へ案内しない
- このフォルダ内のページへアクセスされた場合も、正式入口URLへ転送する
- 今後の本番更新は `boss-team1129/Kimikea-Connect` の `main` ブランチへ反映する
- 公開URLを変えず、キャッシュ更新やファイル更新で対応する

## iPhoneのホーム画面へ追加する手順

1. Safariで `https://boss-team1129.github.io/Kimikea-Connect/` を開く
2. 共有ボタンを押す
3. 「ホーム画面に追加」を選ぶ
4. 名前が `Kimikea Connect` になっていることを確認して追加する
